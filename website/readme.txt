To install, simply copy everything in the root of your website.
It's just 3 html pages + css + javascript

no need to install plugins

please, configure your virtual sky by editing the file:

   js/config.js

if you want, to may further personalize your website

structure:
/js = javascript folder. also contains virtualsky's js files
/img = folder containing the allskycam.jpg 
/video = folder containing the timelapses (2 mp4)
/startrail = folder containing the startrail.jpg 
/css = folder containing the single css file
